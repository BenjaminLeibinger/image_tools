# Drupal 8

With this module is it possible to convert and resize images with the aim for better performance. For this the module provides currently two drush commands.

The first lets you convert uploaded PNG Images to JPG that they can be optimized with other tools. PNG-Files with transparency will be excluded.

With the second command you can resize all the images to an given max width, default is 2048. Here its possible to include PNG-Files. If the have transparency the Background will be set to white.
